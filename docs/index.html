CDot is an object oriented language that was designed to feel familiar to everyone who already knows a language of the C family or any other modern language, such as Swift or Rust.

CDot programs are statically compiled and type-safe, which means that the types of every variable you declare need to be known at compile-time, which differentiates it from dynamic languages like Python and JavaScript. To ease the burden on developers, CDot supports powerful type inference, which means that you won‚Äôt have to write many type annotations in practice.
